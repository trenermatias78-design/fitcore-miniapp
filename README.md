# FitCore Mini App

Telegram Mini App для FitCore бота.

## Локальний запуск

```bash
npm install
npm run dev
```

## Deploy на Railway

Цей проект автоматично деплоїться через GitHub → Railway.

## Структура

- `src/FitCoreApp.jsx` — основний компонент (клієнт + адмін)
- `src/main.jsx` — точка входу + Telegram WebApp init
- `index.html` — HTML шаблон з Telegram SDK
